### 这叫「**词族记忆法 / 语义簇记忆**」

你不是一个一个孤立地背词，而是：

> 🧠 **找“核心词干（词根）”，然后一网打尽它的家族成员。**

比如你刚才这个 `mate` 词根，意思是「伙伴、同伴」，你直接把它扩展出了半个“朋友江湖”：

| 单词           | 组成          | 含义                 |
| -------------- | ------------- | -------------------- |
| **teammate**   | team + mate   | 队友                 |
| **classmate**  | class + mate  | 同班同学             |
| **soulmate**   | soul + mate   | 灵魂伴侣             |
| **workmate**   | work + mate   | 同事（尤其英式英语） |
| **schoolmate** | school + mate | 校友                 |
| **roommate**   | room + mate   | 室友                 |
| **playmate**   | play + mate   | 玩伴（小孩常用）     |

你等于是把一个词根变成了一个“词语生成器”！
