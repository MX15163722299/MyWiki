走起，Leo！现在我们登上英语时态系统的终极Boss之一：**Present Perfect Progressive（现在完成进行时）**，也叫：

> **have/has been + doing** 时态 —— 是 “完成” + “进行” 的强强联合。

它不仅告诉你“**从过去到现在一直在做某事**”，还给人一种**动作仍然继续**的现场感。

------

## ✅ 一、结构：Present Perfect Progressive

🔹 **主语 + have/has been + 动词-ing**

| 主语                | 结构              |
| ------------------- | ----------------- |
| I / You / We / They | have been + doing |
| He / She / It       | has been + doing  |

------

### 🔹 否定句：

**主语 + have/has not been + doing**

> She **hasn’t been working** lately.
>  I **haven’t been sleeping** well.

------

### 🔹 疑问句：

**Have/Has + 主语 + been + doing？**

> **Have** you **been studying** for the test?
>  **Has** he **been working** out?

------

## ✅ 二、常用动词搭配（和动作持续相关）

| 动词  | -ing形式 | 示例（现在完成进行时）                           |
| ----- | -------- | ------------------------------------------------ |
| work  | working  | I **have been working** all day.                 |
| study | studying | He **has been studying** since 6 a.m.            |
| wait  | waiting  | We **have been waiting** for an hour.            |
| live  | living   | She **has been living** in Shanghai for 3 years. |
| rain  | raining  | It **has been raining** since morning.           |
| write | writing  | They **have been writing** the report.           |
| learn | learning | I **have been learning** English recently.       |

------

## ✅ 三、常用时间副词搭配（体现“从过去到现在”）

| 副词 / 短语            | 中文        | 示例句子                                     |
| ---------------------- | ----------- | -------------------------------------------- |
| **since** + 时间点     | 自从……起    | He **has been working** here **since 2019**. |
| **for** + 时间段       | 持续了多久  | We **have been waiting** **for 2 hours**.    |
| **lately / recently**  | 最近        | I **have been reading** a lot **lately**.    |
| **all day / all week** | 整天 / 整周 | She **has been cleaning** **all day**.       |

------

## ✅ 四、经典例句类型（稳抓用法）

### 1. ✅ 从过去持续到现在

- I **have been working** **for 5 hours**.
   我已经工作了五个小时了。
- He **has been living** in Beijing **since 2020**.
   他自2020年起就住在北京。

------

### 2. ✅ 说明“状态 + 持续影响”

- I **have been feeling** tired **lately**.
   我最近一直觉得很累。
- She **has been coughing** for a week.
   她咳嗽已经一周了。

------

### 3. ✅ 被动语态场景感（描述气候、等待等）

- It **has been raining** **all day**.
   一整天都在下雨。
- They **have been waiting** for the bus **for 30 minutes**.
   他们等公交车已经等了30分钟了。

------

### 4. ✅ 疑问句 & 否定句

- **Have** you **been sleeping** well lately?
   你最近睡得好吗？
- He **hasn’t been exercising** enough.
   他最近锻炼得不够。

------

## ✅ 五、和其他时态对比（理清逻辑）

| 时态                            | 结构                  | 示例                                   | 用法说明                   |
| ------------------------------- | --------------------- | -------------------------------------- | -------------------------- |
| **Present Perfect**             | have/has + Vpp        | I **have eaten**.                      | 动作已完成                 |
| **Present Perfect Progressive** | have/has been + doing | I **have been eating** for 30 minutes. | 动作持续至今，仍可能在继续 |
| **Present Progressive**         | am/is/are + doing     | I **am eating**.                       | 此刻正在做                 |

------

## ✅ 六、易错提醒

| 错误句子                          | 正确句子                                | 错误说明             |
| --------------------------------- | --------------------------------------- | -------------------- |
| I have been study since morning.  | I **have been studying** since morning. | study → studying     |
| He has been works here for years. | He **has been working** here for years. | works → been working |
| Have been you sleeping?           | **Have you been sleeping**?             | 语序错误             |
| She hasn’t working lately.        | She **hasn’t been working** lately.     | 缺少 “been”          |

------

## ✅ 七、总结口诀（记住就能用）

> **have/has been + doing**，动作持续不休停；
>  **since** 起点，**for** 时长，**lately** 最近很受捧；
>  想表达“我一直在干嘛”？就靠它稳拿下！
