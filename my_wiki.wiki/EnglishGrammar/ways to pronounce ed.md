##ways to pronounce "ed"

|         |  Id  |  t   |  d   |
| :-----: | :--: | :--: | :--: |
| wanted  |  1   |      |      |
| talked  |      |  1   |      |
| called  |      |      |  1   |
| watched |      |  1   |      |
| decided |  1   |      |      |
| opened  |      |      |  1   |

是的，**规则动词过去式（-ed）** 的发音是有规律的，主要有 **/t/、/d/、/ɪd/** 三种发音方式。具体规则如下：

------

### **1. /t/ 发音**

**当动词词尾是清辅音（** **/p/, /k/, /f/, /s/, /ʃ/, /tʃ/** **等）时，-ed 读作 /t/**。
 📌 **例子**：

- **stopped** /stɒpt/（停止）
- **looked** /lʊkt/（看）
- **laughed** /læft/（笑）
- **kissed** /kɪst/（亲吻）
- **washed** /wɒʃt/（洗）
- **watched** /wɒtʃt/（观看）

------

### **2. /d/ 发音**

**当动词词尾是浊辅音（** **/b/, /g/, /v/, /z/, /ʒ/, /ð/, /l/, /m/, /n/, /r/** **等）时，-ed 读作 /d/**。
 📌 **例子**：

- **lived** /lɪvd/（居住）
- **opened** /ˈoʊpənd/（打开）
- **cleaned** /kliːnd/（清洁）
- **loved** /lʌvd/（爱）
- **played** /pleɪd/（玩）
- **called** /kɔːld/（打电话）

------

### **3. /ɪd/ 发音**

**当动词词尾是 /t/ 或 /d/ 时，-ed 读作 /ɪd/（增加一个音节）**。
 📌 **例子**：

- **wanted** /ˈwɒntɪd/（想要）
- **needed** /ˈniːdɪd/（需要）
- **decided** /dɪˈsaɪdɪd/（决定）
- **hated** /ˈheɪtɪd/（讨厌）
- **invited** /ɪnˈvaɪtɪd/（邀请）

------

### **总结记忆方法**

- **/t/** → 词尾是清辅音（无声），如 **k, p, s, f, ch, sh**
- **/d/** → 词尾是浊辅音（有声），如 **b, g, v, z, m, n, l, r**
- **/ɪd/** → 词尾是 **t 或 d**，需额外加音节
